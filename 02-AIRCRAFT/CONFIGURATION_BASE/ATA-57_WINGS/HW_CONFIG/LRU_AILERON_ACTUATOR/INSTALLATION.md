## 1. Scope

This manual provides installation procedures for the Aileron Actuator on [Aircraft Model].

**Function**: Roll control actuation  
**Location**: Wing aileron hinge line

## 2. Safety Precautions

⚠️ **WARNING**: Follow all safety procedures before beginning installation.

- Ensure hydraulic/electrical systems are depressurized/de-energized
- Use proper personal protective equipment (PPE)
- Follow lockout/tagout procedures
- Verify proper tooling and torque wrenches are calibrated

## 4. Installation Procedure

### 4.1 Pre-Installation Inspection
1. Verify LRU part number matches installation requirements
2. Inspect LRU for shipping damage
3. Verify all mounting hardware is present
4. Review [Specifications](SPECIFICATIONS.md) for interface requirements

### 4.2 Removal of Old Unit (if applicable)
1. Disconnect electrical connectors - Reference [ATA-92 EWIS](../../ATA-92_EWIS/)
2. Disconnect hydraulic lines (if applicable)
3. Remove mounting hardware
4. Carefully extract unit from installation position

### 4.3 Installation of New Unit
1. Position new LRU in mounting location
2. Install and torque mounting hardware per specification
3. Connect hydraulic lines (if applicable) - Reference [ATA-29](../../ATA-29_HYDRAULIC_POWER/)
4. Connect electrical connectors - Reference [Connector Documentation](../CONNECTORS/CONTROL_SURFACE_CONNECTORS.md)
5. Verify all connections are secure

### 4.4 Post-Installation Checks
1. Perform visual inspection of installation
2. Verify clearances and interference
3. Conduct leak check (if applicable)
4. Perform functional test per [Test Procedures](TEST_PROCEDURES.md)
